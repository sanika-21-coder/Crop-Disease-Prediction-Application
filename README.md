# 🌱 CropScan AI - Intelligent Crop Disease Predictor & Agri-Tech Platform

### 🚀 **Live Demo: [cropscan-ai.onrender.com](https://cropscan-ai.onrender.com)**
*(Note: As this is hosted on a free tier, it may take about 50 seconds to "wake up" if it hasn't been visited recently).*

Welcome to **CropScan AI**, a comprehensive Project-Based Learning (PBL) application developed for Semester 6. 

This project bridges the gap between Deep Learning and modern web development, providing farmers and agricultural workers with an all-in-one platform to diagnose crop diseases, get treatment recommendations, and purchase agricultural products.

## 🚀 Key Features

* **High-Accuracy AI Diagnosis**: Uses a custom-trained **MobileNetV2** Convolutional Neural Network (CNN) to instantly identify 38 different classes of crop diseases and healthy leaves.
* **Intelligent Chatbot**: A context-aware, local AI chatbot that offers custom crop recommendations based on soil type and temperature, and answers specific questions about diagnosed diseases.
* **Premium E-Commerce Marketplace**: A fully functional shop with 15+ agricultural products (Seeds, Fertilizers, Pesticides), complete with dynamic filtering, 3D tilt interactions, and a sliding Shopping Cart.
* **Modern UI/UX**: Designed with a stunning, dark-themed Glassmorphism aesthetic, interactive 3D flashcards, and a custom water-bubble cursor effect.

## 🛠️ Technology Stack

* **Machine Learning**: TensorFlow, Keras, NumPy
* **Backend**: Python, Flask, Werkzeug
* **Frontend**: HTML5, CSS3 (Glassmorphism), Vanilla JavaScript
* **Database**: Local JSON mapping (`disease_info.json`)
* **Deployment Ready**: Configured for Render/Gunicorn.
